Concept: Primitive vs Reference

  • Primitive = Copy by Value
  • Object = Copy by Reference
  • Shallow Copy


      let a = 10;
      let b = a;
      b = 20;
      console.log(a); // 10
      console.log(b); // 20
    


      let obj1 = { value: 10 };
      let obj2 = obj1;
      obj2.value = 20;
      console.log(obj1.value); // 20
      console.log(obj2.value); // 20
    


      let obj1 = { value: 10 };
      let obj2 = { ...obj1 }; // create a shallow copy
      obj2.value = 20;
      console.log(obj1.value); // 10
      console.log(obj2.value); // 20

Concept: Functions & Scope

Core Concepts:

  • Functions are logic containers: input → process → output.
  • Scope defines where variables “live” and who can “see” them.
  • Encapsulation means each function protects its internal variables unless explicitly returned.


      Global Scope
        └─ outer() Scope
          └─ inner() Scope

Concept: Open Closures

A closure is when a function “remembers” the variables from its outer scope, even after that outer function has finished running.


      Global Scope
      └─ outer() runs → creates count = 0
          └─ inner() closes over count
              ↓
              Assigned to counter
              ↓
              counter() can access count forever
    


      function outer() {
        let count = 0;

        return function inner() { 
          count++;
          console.log(`Count is: ${count}`);
        };
      }
      const counter = outer(); // ← ✅ This calls `outer()` ONCE and stores the returned `inner()` function in `counter
      counter(); // ❗ Only calls `inner()` — NOT outer()
      counter(); // ❗ Again, only calls `inner()`, NOT outer()
