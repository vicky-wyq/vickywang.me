Main goal

The goal of this navigation update is to present the design thinking process in a clear and logical manner.

Breakdown

  • Able to understand the context easily
  • easily determine where the user is looking/reading
  • skip or jump to the sections they are interested in
  • revisiting previous sections if desired

Version 1

The main goal of the initial version was to improve the readability, scannability, scrollability, and skippability of the UX case study projects.

Version 2

After receiving feedback and conducting testing, the goal for the iteration was to streamline the content and ensure that users can find what they're looking for with minimal effort.

Version 3

After conducting additional usability testing, I have refined the method of displaying content in a single location, which is more organized and avoids the need for a lengthy list.
