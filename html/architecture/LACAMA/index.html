The Problem Cloud

PROBLEM CLOUD

Using the skills, tools and materials gathered to date including folding, laser cutting, assembling layers of plywood, manipulating images into vectors to construct a layered assemblage the problem cloud.

PROBLEM CLOUD

2 images in 3 color separations and one monochrome image. Take into account folding as studied in the box assignment, as well as offsetting to deal with corner and ground conditions.

The Cloud

PROBLEM CLOUD DIAGRAM

2 images in 3 color separation and one monochrome image are interleaved in a repetitive pattern.
7 Layers total, Spacing: 1/4”
Monochrome image can be located anywhere within this layering.

OPERATION METHOD

an armature continues around the corner for structure, image layers are offset perpendicular from the innermost layer with no distortion or shifting

OPERATION METHOD

no corner armature, image layers are offset perpendicular with no distortion or shifting

OPERATION METHOD

no corner armature, image layers are offset perpendicular from the outermost layer with no distortion or shifting, and then are trimmed

Color & Layers

FIRST ROUND TEST

five sides of a box unfolded. As one layer of the image is folded to the box, formerly contiguous color areas are disconnected. An armature is necessary to hold the image layer together, and to align it with the other image layers.
